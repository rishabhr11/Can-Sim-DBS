# _CAN-DBC Message Simulator_
Simulate CAN messages by loading DBC files for automated testing.

## Getting Started
The simulator supports CAN .dbc parsing, simulated messages transmission and received messages interpretation.<br>
Features:
1. Loading and parsing .DBC files
1. Simulating CAN message transmission
1. Interpreting received CAN messages



## External libraries used

1. CAN Bus manager<br>
`pip install python-can`
1. CAN .DBC Parser<br>
`pip install cantools`
1. Tkinter Module<br>
`pip install tk`
1. Pandas Module<br>
`pip install pandas`
1. Pandas Tabel Module<br>
`pip install pandastable`

## Tools and drivers
Used PEAK CAN adapter with Busmaster.<br>
1. https://www.peak-system.com/Drivers.523.0.html?&L=1 (Peak CAN driver)
1. https://rbei-etas.github.io/busmaster/download.html (Busmaster)
